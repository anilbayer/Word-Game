# Word Masters - A Fun Word Guessing Game

Word Masters is an engaging word-guessing game inspired by the popular Wordle. The game challenges players to guess a 5-letter word within six attempts, providing feedback on each guess to help refine their choices.

## Features
- **Dynamic Word Generation**: Fetches a new "word of the day" every day from an external API.
- **Interactive Feedback**:
  - Correct letters are highlighted in green.
  - Letters in the word but in the wrong position are marked in yellow.
  - Incorrect letters are shaded gray.
- **Keyboard Input**: Players can use their keyboards to input guesses.
- **Responsive Design**: Works seamlessly on various devices and screen sizes.
- **Animations**: Eye-catching animations for invalid words and victory celebrations.

## Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Edge, etc.)
- A working internet connection to fetch the word of the day and validate guesses.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/word-masters.git
   ```
2. Navigate to the project directory:
   ```bash
   cd word-masters
   ```

### Running the Game
1. Open `word-game.html` in your browser.
2. Start guessing the word of the day!

## How to Play
1. Enter your guess using the keyboard.
2. Press `Enter` to submit your guess.
3. Get feedback:
   - Green: Correct letter in the correct position.
   - Yellow: Correct letter but in the wrong position.
   - Gray: Incorrect letter.
4. Use the feedback to refine your guesses and try to guess the word in six attempts!

## Project Structure
- **`word-game.html`**: The main HTML file for the game.
- **`word-game.js`**: The JavaScript logic for the game, including fetching words, handling user input, and displaying feedback.
